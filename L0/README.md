# L0 Package

L0 contains direct telemetry/data-read SDK cards.

## Cards

| function | English | Chinese | Bilingual |
|---|---|---|---|
| `Agentech.get_battery_status` | `13_get_battery_status.en.md` | `13_get_battery_status.zh.md` | `13_get_battery_status.bilingual.md` |

## Boundary

L0 cards read state only. They do not command motion, stream over time, estimate runtime, decide execution safety, schedule charging, or plan tasks.
